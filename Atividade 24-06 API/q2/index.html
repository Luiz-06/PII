Showcase de APIs Públicas

1. Open Notify (Posição da ISS)

A API Open Notify é um serviço de código aberto que fornece dados sobre a Estação Espacial Internacional (ISS). É uma API pública e gratuita que não requer autenticação, ideal para projetos simples e para quem quer começar a consumir dados de um serviço externo.

Demonstração via cURL:

curl http://api.open-notify.org/iss-now.json

2. The Movie Database (TMDb)

A TMDb é uma das maiores e mais populares fontes de dados sobre filmes e séries. Seu uso é gratuito para projetos não comerciais, mas requer uma chave de API para autenticar as requisições. Essa chave funciona como um token que identifica sua aplicação.

Atenção: Para usar o comando abaixo, obtenha sua chave gratuita no site da TMDb e substitua o texto SUA_CHAVE_API_AQUI. Nunca publique sua chave em repositórios públicos!

Demonstração via cURL:

curl "https://api.themoviedb.org/3/movie/550?api_key=SUA_CHAVE_API_AQUI&language=pt-BR"

3. Universities List

Esta é uma API pública e gratuita que fornece uma lista de universidades de diversos países. É uma ferramenta útil para projetos educacionais ou para aplicações que precisam de dados sobre instituições de ensino superior. A busca pode ser filtrada por país.

Demonstração via cURL:

curl "http://universities.hipolabs.com/search?country=Brazil"

4. QR Code Generator

Uma API extremamente simples e direta para gerar QR Codes. Ela não requer autenticação e funciona através de uma requisição GET. Os parâmetros para a geração do QR Code são passados diretamente na URL e a API retorna a imagem correspondente.

Demonstração via cURL:

curl "https://api.qrserver.com/v1/create-qr-code/?size=150x150&data=GitHub-Pages" --output qrcode.png

Este comando não retorna texto no terminal, mas cria um arquivo de imagem chamado qrcode.png no seu diretório.

Página de demonstração criada em 27 de junho de 2025.
